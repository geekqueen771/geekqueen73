# Social Media Learning App

## Project Overview
The Social Media Learning App is designed to facilitate learning through social media platforms. It leverages users' familiarity with social media to enhance educational experiences.

## Features
- **User-friendly Interface**: Easy navigation based on popular social media platforms.
- **Interactive Learning Modules**: Engaging content that encourages participation and retention.
- **Social Sharing**: Users can share their achievements and progress on social media to motivate others.

## Technologies Used
- **Frontend**: ReactJS, CSS, HTML
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Getting Started
1. Clone the repository: `git clone https://github.com/geekqueen771/geekqueen73`
2. Navigate to the project directory: `cd geekqueen73`
3. Install dependencies: `npm install`
4. Start the application: `npm start`
